02 / PRACTICE REALITY

Browser Lessons, Terminal Runs

Use the page to read examples and diagrams. Use Ubuntu Terminal to install OpenJDK with sudo apt, create files, compile with javac, and run with java.

1 Install Java tools 
sudo apt update
sudo apt install openjdk-21-jdk
2 Create a small file 
mkdir -p java-practice
cd java-practice
nano HelloJava.java
3 Compile it 
javac HelloJava.java
4 Run the compiled class 
java HelloJava
