# Glossary — CLI Tool

| Term | Definition |
|------|------------|
| **argv** | Command-line arguments array (`process.argv`) |
| **bin** | `package.json` field mapping command names to scripts |
| **CLI** | Command Line Interface |
| **CommonJS** | Node module system using `require` / `module.exports` |
| **Dependency** | Package your project needs to run |
| **devDependency** | Package only for development (tests, lint) |
| **ESM** | ECMAScript modules using `import` / `export` |
| **Exit code** | Numeric status when program ends (0 = success) |
| **fs module** | Node built-in for file read/write |
| **Global install** | `npm i -g` — command available system-wide |
| **Local install** | `npm i pkg` — package in project's node_modules |
| **npm** | Node Package Manager |
| **npm link** | Symlink local package for global testing |
| **npx** | Run package binary without global install |
| **path module** | Node built-in for file path joining |
| **process** | Global object — env, argv, exit |
| **Registry** | npm's package server (registry.npmjs.org) |
| **SemVer** | Semantic versioning MAJOR.MINOR.PATCH |
| **Shebang** | `#!` line telling OS which interpreter to use |
| **stdin** | Standard input stream |
| **stdout** | Standard output (console.log) |
| **stderr** | Standard error (console.error) |

See [concepts.md](../concepts.md), [Node.js](../../../docs/nodejs/README.md), [npm](../../../docs/npm/README.md).
