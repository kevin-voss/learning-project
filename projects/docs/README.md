# Learning library

Topic docs organized by folder. **You do not read this cover to cover.**

## How learning works here

1. Open the **[Roadmap](./roadmap.md)** and start **Project 1**.
2. Each phase tells you **what to build** and **which topic pages to read** (usually 5–15 minutes).
3. Read → apply in code → commit → next phase.
4. Come back to this library when a new tool appears (Docker in Project 2, npm in Project 3, React in Project 4).

```
Roadmap phase  →  Read 1–2 topic pages  →  Build that phase  →  Repeat
```
